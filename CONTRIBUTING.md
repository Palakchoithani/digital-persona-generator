# Contributing


Thanks for wanting to contribute! This guide is intentionally short and beginner-friendly.


## Getting started


1. Fork this repository.
2. Clone to your machine:
```bash
git clone https://github.com/<your-username>/digital-persona.git
cd digital-persona/web
```

3. Install dependencies and run dev server:
```bash
npm install
npm run dev
```

## Find a task
- Start with issues labeled easy.
- If you want to work on a task, comment on the issue "I want to work on this".

## Branching

Create a branch for your work:
```bash
git checkout -b feat/short-description
```

## Make changes & tests

- Keep commits small and focused.
- Update or add tests where applicable.

## Open a Pull Request

- Push your branch to your fork.
- Open a PR to main and pick the appropriate template.
- In the PR description reference the issue you're solving.

## Code style

- Use consistent formatting (Prettier/ESLint recommended).
- Keep code readable and well-commented for beginners.

## Need help?

- Ask in the issue thread — include what you tried and errors OR you can ask on ACWOC discord server.
- Maintain a short comment like: "I'm stuck at X — here's what I tried..."

> Thank you! Your contributions make this project better.
