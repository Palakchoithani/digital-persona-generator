# Contributing
Thanks for contributing! This repo is event-friendly. Follow these steps:
## Code of conduct
Please read `CODE_OF_CONDUCT.md`.
## Setup
1. Fork & clone
2. `cd web && npm install`
3. `npm run dev`

## Branching & PRs
- Create a feature branch: `git switch -c feat/short-desc`
- Commit atomic changes with clear messages.
- Open PR against `main` with the `PULL_REQUEST_TEMPLATE.md`.
## Issues
- Pick `good-first-issue` or `help-wanted` labeled tasks.
- If you need mentorship, add a comment tagging `@maintainer`.
## Communication
- Use GitHub discussions or join the project's Slack/Discord (link in README).
## Mentorship
- Mentors will add review comments and optionally pair-program via livestream or
calls.
